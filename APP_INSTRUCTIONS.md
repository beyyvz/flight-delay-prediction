# Flight Delay Prediction App — Easy Step‑By‑Step Instructions 🚀

Follow each step in order. You can copy and paste every command directly into your terminal.
*Using Visual Studio Code*
---

## 1. Install the required packages

Copy and paste this:

**pip install streamlit scikit-learn numpy pandas joblib matplotlib**

If you use a specific Python version, copy this instead:

**path\to\python.exe -m pip install streamlit scikit-learn numpy pandas joblib matplotlib**

---

## 2. Train the model (run this once)

Copy and paste this:

**python train_model.py**

If you use a specific Python version, copy this:

**path\to\python.exe train_model.py**

This will create the file:

**flight_delay_model.pkl**

---

## 3. Run the Streamlit app

Copy and paste this:

**streamlit run app.py**

Your browser will open automatically. If it doesn’t, copy the link shown in the terminal into your browser.


---

## Notes

- This model uses synthetic data and is for demonstration only.
- Predictions should not be used for real-world flight decisions.
- Real delays depend on weather, staffing, maintenance, and air traffic.
